Predicting Apartment Price with Machine Learning Models

This project focuses on predicting apartment prices using the "inmuebles.json" dataset. It involves preprocessing the data, applying logarithmic transformations, and encoding categorical variables. Three models—linear regression, decision trees, and random forests—are trained and evaluated based on their R^2 scores. Cross-validation and random search are used to enhance model performance and optimize hyperparameters. The aim is to determine the most accurate model for predicting apartment prices.

File Metadata Microservice

This project is a file metadata microservice built with Express, Multer, and CORS. It allows users to upload a file through a form, analyzes the file, and returns its metadata, such as the file name, type, and size. Multer is used for handling file uploads, with a file size limit of 10MB. The service exposes an API endpoint (/api/fileanalyse) to upload a file and receive the file metadata in a JSON response.

This project is a web application built with Express that provides a date converter API. It includes a homepage that serves a static HTML file. The API endpoint /api/:date? allows users to get the current date or convert a given date or Unix timestamp into both Unix and UTC formats. If the input is invalid, it returns an error message. Url Shortener

This project is a URL shortener service. It provides two main functionalities: shortening URLs and redirecting short URLs to their original addresses. Users can submit a URL via a POST request to the /api/shorturl endpoint, and if the URL is valid, the service will generate a short URL identifier and return it along with the original URL. Users can then access the original URL by sending a GET request to /api/shorturl/:shorturl with the short URL identifier, which will redirect them to the original URL. The project uses Express for the server and maintains a list of URLs and their corresponding short identifiers in memory.

Roman Numeral Converter

This project is a web application that converts decimal numbers into Roman numerals. Users can enter a number in a text field, and by clicking a button, the number is converted to its Roman numeral representation. The application validates that the entered number is within the allowed range (from 1 to 3999) and displays error messages if the number is invalid or out of range. The conversion is handled by a function that uses a table of equivalences between decimal values and their corresponding Roman numeral symbols.
